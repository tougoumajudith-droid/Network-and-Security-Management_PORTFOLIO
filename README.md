# Network Security — Final Skills Assessment
Cisco Packet Tracer · Routers 4331/4221 · Catalyst 2960

## Overview
Final practical assessment for a Network Security course. Covers the full 
design and implementation of an enterprise network across four parts.

## Network Topology
Three-site architecture: CAMPUS (MTL) — BRANCH (TOR) — INTERNET (ISP)

## Part 1 — Core Network Services
Switching, Routing (OSPF), DHCP, NTP, NAT/PAT, DNS, HTTP, TFTP.

## Part 2 — Security Hardening
SSH, AAA with RADIUS, SYSLOG, NTP authentication, OSPF MD5 authentication,
port security, DHCP snooping, Dynamic ARP Inspection, STP PortFast, 802.1X.

## Part 3 — Zone Policy Firewall
ZPF on BRANCH router inspecting HTTP, HTTPS, DNS, ICMP, FTP, Telnet, SMTP, POP3.

## Part 4 — Site-to-Site IPsec VPN
IPsec tunnel between MTL and TOR using AES-256, SHA, DH group 5.

## Tools
Cisco Packet Tracer · IOS CLI · OSPF · RADIUS · IPsec · 802.1X
